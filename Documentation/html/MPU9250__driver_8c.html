)

Set the IMU address, check for connection, reset IMU, and set full range scale.

Parameters
I2CxPointer to I2C structure config.
addrHex address based on AD0 pin - 0x68 low or 0x69 high.
aScaleSet accelerometer full scale range: 0 for ±2g, 1 for ±4g, 2 for ±8g, and 3 for ±16g.
gScaleSet gyroscope full scale range: 0 for ±250°/s, 1 for ±500°/s, 2 for ±1000°/s, and 3 for ±2000°/s.
tauSet tau value for the complementary filter (typically 0.98).
dtSet sampling rate in seconds determined by the timer interrupt.
